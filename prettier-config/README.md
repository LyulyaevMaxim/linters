### How to use
0. Install
   ```
   yarn add --dev @lyulyaev-maxim/prettier-config
   ```
1. Connect to your Prettier configuration:
   ```
   "prettier": "@lyulyaev-maxim/prettier-config"
   ```
2. Run to check your files:
   ```
   prettier --check .
   ```
